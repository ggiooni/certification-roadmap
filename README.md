# 🤖 AI Automation Engineer — Certification Roadmap

Interactive tracker for my certification path: UiPath → UCD Python → Microsoft PL-500.

## Features
- ✅ Checklist with localStorage persistence
- 📅 Google Calendar integration (manual links + API ready)
- 📝 Notes per step
- 📥 Export progress as JSON
- 📊 Progress tracking per phase and total

## Setup
```bash
npm install
npm run dev
```

## Google Calendar API Setup (optional)
1. Go to https://console.cloud.google.com/
2. Create project → Enable "Google Calendar API"
3. Create OAuth 2.0 credentials
4. Update `CALENDAR_CONFIG` in `src/App.jsx`

## Tech Stack
- React 18 + Vite
- localStorage for persistence
- Google Calendar API (optional)
